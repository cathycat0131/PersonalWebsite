Relica Travel Planner, Software Development Project

  • Relica is a travel planning GIS. It has a personalized Trip Planning Feature. It allows travelers to create a customized itinerary. With this feature, users can search up and save multiple locations with custom names, put in notes for each place, and easily pull out their travel routine in simple steps.
  • Used the OpenStreetMap API and C++ programming language
  • Implemented optimization algorithms such as Dijkstra’s, A*, Greedy Algorithm and Simulated Annealing to obtain optimal path in the attempt to solve the Travelling Salesman Problem
  • Designed GUI using the GTK and EZGL graphics packages. Implemented callback functions to enable interactivity of map to search and save desired location via mouse click and keyboard
  • Used matrix and multi-threading to enhance Map’s speed performance and user experience
See Demo Video for searching and saving locations

Movie Poster Classifier, Machine Learning Project

  • A machine-learning model using Convolutional Neural Network, Res-Net 50, and Pytorch that classifies movie poster images by genre
  • Implemented primary Neural Network and the baseline model. Researched the use of confusion matrix to analyze basic performance
  • Trained the CNN model on 12,000 poster images using Res-Net 50 and obtained a final accuracy of over 65% across 8 genres
  • Evaluate the model’s accuracy using testing data and validation data
  • Web application allows users to upload images and view model predictions using a Python Flask REST API
